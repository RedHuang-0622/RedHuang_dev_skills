## 特殊注意事项（聊天/邮件）

1. **多轮对话**：
   - 同一资产可能在多轮对话中逐步补全信息，需要合并
   - 以最后确认的信息为准
   - 如果存在矛盾信息，标注 needs_review=true

2. **口语化表达**：
   - "大概"、"左右"、"差不多" → confidence 降低 0.1
   - "XX万" → 转换为数字 × 10000
   - 简称/缩写 → 尝试展开

3. **发送者识别**：
   - 区分不同发送者，标注信息来源
   - 权威来源（如"评估师"）的 confidence 高于普通来源

4. **时间戳**：
   - 以消息时间戳作为信息时效性参考
   - 超过 30 天的旧消息中的数值信息标注可能已过期

5. **输出格式**：同通用模板，额外增加 `source_sender` 字段

```json
[
  {
    "fields": { "字段名": "值" },
    "confidence": 0.90,
    "field_confidences": { "字段名": 0.92 },
    "source_quote": "原文片段",
    "source_sender": "发送者标识",
    "needs_review": false
  }
]
```
