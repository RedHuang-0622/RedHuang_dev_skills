# 任务：从以下文本中提取资产条目

## 资产类型
{{ asset_type_name }}

## 字段定义
{{ fields_table }}

## 文本
{{ raw_text }}

## 输出要求

请以 JSON 数组格式输出提取结果。每个元素为一个资产条目对象，结构如下：

```json
[
  {
    "fields": {
      "字段名1": "提取的值",
      "字段名2": 数值
    },
    "confidence": 0.95,
    "field_confidences": {
      "字段名1": 0.98,
      "字段名2": 0.85
    },
    "source_quote": "原文中提取该条目的对应文本片段"
  }
]
```

## 要求

1. **完整性**：提取文本中所有可识别的资产条目，不遗漏
2. **准确性**：字段值必须严格来自原文，不推测、不编造
3. **置信度**：
   - 1.0 = 原文直接明示
   - 0.8-0.99 = 可从上下文推断
   - 0.5-0.79 = 需结合常识推测（标记 needs_review=true）
   - <0.5 = 不要输出
4. **原文引用**：source_quote 必须包含原文中对应的文本片段
5. 数值字段去除单位（如"万元"→转换为元），日期统一为 YYYY-MM-DD 格式
6. 如果文本中没有对应字段，不输出该字段（不要填 null 或空字符串）
